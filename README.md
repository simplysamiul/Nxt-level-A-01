## What are some differences between interfaces and types in TypeScript?
```

TypeScript এর মধ্যে type এবং interface অনেকটা একভাবেই কাজ করে। অল্প কিছু পার্থক্য আছে এগুলোর মধ্যে। নিম্নে উল্লেখ করা হলো : 

interface এবং type দুটোই object এর মতো করে উল্লেখ করতে হয় কিন্তু সিনটেক্সট কিছুটা ভিন্ন রকম:

interface User {
  name: string;
}

type User = {
  name: string;
}





```
